Moved Files:
------------
Android Gradle projects use a different directory structure than ADT
Eclipse projects. Here's how the projects were restructured:

* AndroidManifest.xml => app/src/main/AndroidManifest.xml
* assets/ => app/src/main/assets/
* libs/NCL.jar => app/libs/NCL.jar
* libs/armeabi-v7a/lib_nymi_sdk.so => app/src/main/jniLibs/armeabi-v7a/lib_nymi_sdk.so
* libs/armeabi-v7a/lib_nymi_sdk_net.so => app/src/main/jniLibs/armeabi-v7a/lib_nymi_sdk_net.so
* libs/armeabi/lib_nymi_sdk.so => app/src/main/jniLibs/armeabi/lib_nymi_sdk.so
* libs/armeabi/lib_nymi_sdk_net.so => app/src/main/jniLibs/armeabi/lib_nymi_sdk_net.so
